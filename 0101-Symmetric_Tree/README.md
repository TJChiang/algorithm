有一個二元樹 `root` ，確認它是否對稱。

**範例 1**

```
      1
    /   \
   2     2
  / \   / \
 3   4 4   3
```

```
Input: root = [1,2,2,3,4,4,3]
Output: true
```

**範例 2**

```
      1
    /   \
   2     2
    \     \
     3     3
```

```
Input: root = [1,2,2,null,3,null,3]
Output: false
```

**限制邊界**
- 此二元樹的節點數量範圍： [1, 1000]
- -100 <= Node.val <= 100
