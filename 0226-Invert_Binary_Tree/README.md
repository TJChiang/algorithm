有一個 `root` 二元樹，反轉此二元樹，並回傳其根節點

**範例 1**

```
     4                       4
   /   \                   /   \
  2     7        =>       7     2
 / \   / \               / \   / \
1   3 6    9            9   6  3  1
```

```
Input: root = [4,2,7,1,3,6,9]
Output: [4,7,2,9,6,3,1]
```

**範例 2**

```
  2            2
 / \    =>    / \
1   3        3   1
```

```
Input: root = [2,1,3]
Output: [2,3,1]
```

**範例 3**

```
Input: root = []
Output: []
```

**限制邊界**
- 二元樹節點的數量範圍：[0, 100]
- -100 <= Node.val <= 100
