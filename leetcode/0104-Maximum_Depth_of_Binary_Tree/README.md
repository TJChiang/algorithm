有一個二元樹 `root` ，回傳最大深度。

二元樹的`最大深度`是根節點到最遠節點，最長路徑上的節點數。

**範例 1**

```
      1
    /   \
   9     20
        / \
       15  7
```

```
Input: root = [3,9,20,null,null,15,7]
Output: 3
```

**範例 2**

```
   1
    \
     2
```

```
Input: root = [1,null,2]
Output: 2
```

**限制邊界**
- 此二元樹的節點數量範圍： [0, 10000]
- -100 <= Node.val <= 100
